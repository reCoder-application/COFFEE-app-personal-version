・本Webアプリケーションは、日常嗜むスペシャルティコーヒーに関して抽出方法から始まり品種ごとの風味の違い・分析を文字・チャートとして記録すると共に、全国で購入したコーヒーのカードをストックしていくというコレクション性も兼ねている。

・このアプリケーションの意義：サークル活動では主に世界各地のスペシャティコーヒーを用いているが、様々な抽出方法を試してもその時のテイスティングノートは保管されず、記録として残らない現状を打破することを第一。第二に、サークルメンバーのコーヒーそのものに対する好奇心・熱意を、日々自分が飲むコーヒーを他人と共有するという活動を通じて活性化するという狙いがある。

・記録方法に関してはブラウザに搭載されているlocal strageを使用しているが、カード1枚あたりのデータ数はせいぜい500~600バイト程度であるため、使用するブラウザごとの記録可能な数に差はあるが、単純計算でどのブラウザでも10,000個ほどのデータを保存できるため問題としない。

・また、本アプリケーションは別で開発しているミニSNS版のプロトタイプとしての役割も担っている。Local strageを利用するために、他人との記録の比較をアプリケーション上で行うことは本アプリではできない。
