# Coffee App Project
・本Webアプリケーションは日常飲んだコーヒーの味わいの感想・分析内容を他人と共有することを目的としたミニSNSWebアプリケーションである。

・大学のサークルメンバーが主なターゲットとなるアプリケーションであるが、その限りではない。

・このアプリケーションの意義：サークル活動では主に世界各地のスペシャティコーヒーを用いているが、様々な抽出方法を試してもその時のテイスティングノートは保管されず、記録として残らない現状を打破することを第一。第二に、サークルメンバーのコーヒーそのものに対する好奇心・熱意を、日々自分が飲むコーヒーを他人と共有するという活動を通じて活性化するという狙いがある。
