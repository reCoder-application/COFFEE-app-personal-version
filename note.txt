JSON.stringify:Javascriptのオブジェクトや値を、JSON文字列に変換するためのメソッド
基本的な使い方↓
const obj = {name: "Kevin", age: 23, city: "NewYork"}
const jsonString = JSON.stringify(obj)

console.log(jsonString)
結果:　{"name": "Kevin", "age": "23", "city": "NewYork"}


classList
特定の要素のクラス名を追加したり、削除したり、参照したりすることができるメソッド。

sコールバック関数：
「あとで実行してね」と渡す関数。
身近な例だと、addEventListener
modalCancelBtn.addEventListener('click', function() {
    closeModal();
});
ここで、function() {closeModal();}がコールバック関数。



アロー関数
書式は以下の通り
(引数, ...) => {処理}
このとき、引数は無くてもok。


filterメソッド
配列の要素を特定の条件で絞り込み、新しい配列を作る。
例↓
const numbers = [1,2,3,4,5]
const evenNumbers = numbers.filter(num => num % 2 === 0);
